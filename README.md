# Hackathon Frontend Template

A React 19 + TypeScript frontend starter with Vite, TanStack Router, MUI, and form/validation tooling.

## Tech Stack

| Category          | Choice                                            |
| ----------------- | ------------------------------------------------- |
| **Runtime**       | React 19, TypeScript 5.9                          |
| **Build**         | Vite 7, SWC (React)                               |
| **Routing**       | TanStack Router (file-based, auto code-splitting) |
| **UI**            | MUI (Material UI) 7, Emotion                      |
| **Forms**         | React Hook Form, Zod, `@hookform/resolvers`       |
| **HTTP**          | Axios                                             |
| **Feedback**      | react-hot-toast                                   |
| **Lint / Format** | ESLint 9 (flat config), Prettier                  |
| **Git**           | Husky + lint-staged (pre-commit)                  |

- **Routing**: File-based with TanStack Router. Add routes under `src/routes/`; `routeTree.gen.ts` is regenerated by the Vite plugin.
- **Path alias**: `@/*` → `src/*` (see `tsconfig.app.json` and `vite-tsconfig-paths`).

## Theming

- **Palette**: Custom MUI palette in `src/styles/palette.ts` (primary indigo, secondary slate, semantic + grey/blue/green).
- **Typography**: Rubik-based theme in `src/styles/typography.ts`.
- Theme is applied via `ThemeProvider` in `src/providers/ThemeProvider.tsx`, used in the root layout.

## CI

- **Workflow**: `.github/workflows/ts-compiler.yml`
- **Trigger**: On pull requests
- **Steps**: Checkout → Node 20 → `npm ci` → `tsc -b`
